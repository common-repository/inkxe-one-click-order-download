== Description ==

It is used to download order for inkXE cusomize product only.
After installing this plugin a button having name **Download** will be added at Order listing section 
on **Admin Woocommerce Orders** page.

= Main features =
* **Simplest way to download the orders**
* One click download our orders.
* If product is not avaliable then notify users.
* **No restrictions of order status. Order with any status can be downloaded**.
* Easy customization.
* Easy to use and install.
* Easy to configure.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of inkXE One Click Order Download , log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "inkXE One Click Order Download" and click Search Plugins. Once you've found our inkXE One Click Order Download plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our One Click Order Re-Order plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
